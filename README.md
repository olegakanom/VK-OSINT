## Инструменты мониторинга и слежки

|----|----------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| 1  | @AximoBot                                                | Мгновенно сохраняет новые публикации аккаунта ВК (уведомления о постах)                            |
| 2  | https://220vk.top/                                       | Слежка за аккаунтом: после добавления — аватары, лайки, комментарии, друзья, группы и т.д.          |

Советы:
- Многие сервисы (особенно x-ray, 220vk и боты) требуют VPN из-за блокировок в РФ.
- Для графов друзей используй только публичные профили — приватные не парсятся.
- Проверяй актуальность ботов: VK часто режет API, поэтому некоторые инструменты теряют функционал со временем.
